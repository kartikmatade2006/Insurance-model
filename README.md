#Insurance-model
-This is my first Machine Learning project. I have built a simple Linear Regression model that predicts the insurance price for a person based on basic information such as age, gender, BMI, number of children, smoking status, and region.

The model was trained on a dataset containing records of 10,000 individuals.

Features used: age, gender, BMI, children, smoker/non-smoker, and region.

Implemented using Python (scikit-learn) and deployed with Flask.

The web interface allows users to input their details and get a predicted insurance price instantly.
